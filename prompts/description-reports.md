# Description Reports
This document lists notable descriptive choices and language from each scene that has already been written. As you write new scenes, use what is here to achieve 2 results:

1. consistency: don't describe a given aspect of the miliue in contradictory ways; cycad mash can't smell both "moldy" and "yeasty"
2. freshness: don't redundantly redescribe the same interesting details in the milieu, scene after scene -- and don't  use the same words or phrases as descriptors.
